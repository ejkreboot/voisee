# voisee Landing Page

## What Changed

The landing page has been completely redesigned with a professional marketing approach:

### New Sections

1. **Hero Section**
   - Compelling headline emphasizing family connection
   - Clear value proposition
   - Two CTAs: "Connect Now" (scrolls to connection form) and "Setup Kiosk Mode"
   - Background image from your assets

2. **Features Section**
   - 6 feature cards highlighting key benefits
   - Focus on simplicity, security, and family connections
   - Responsive grid layout

3. **How It Works**
   - 3-step visual guide
   - Clear, simple explanations
   - Numbered steps with icons

4. **Use Case / Testimonial**
   - Real-world example (grandmother connection)
   - Emotional appeal with background image
   - Helps prospects envision using the product

5. **Connect Section**
   - The actual connection form (room name + password)
   - Positioned as a CTA after marketing content
   - Clean, focused design

6. **Footer**
   - Brand reinforcement
   - Professional closing

### Design Features

- Professional color palette (maintained from original)
- Responsive design (works on mobile, tablet, desktop)
- Smooth scrolling and transitions
- Modern, clean aesthetic
- Family-focused messaging

### User Flow

1. Visitor lands on hero → sees value proposition
2. Scrolls through features → understands benefits
3. Reads how it works → knows what to expect
4. Sees testimonial → builds trust
5. Reaches connection form → takes action

### Key Selling Points to Emphasize:

1. **Zero Tech Skills Required** - Perfect for seniors/grandparents
2. **Always Available** - No scheduling needed, just call
3. **Auto-Answer** - Kiosk mode means no buttons to press
4. **Secure** - Password-protected rooms
5. **No Apps** - Works in any web browser
6. **Perfect for Family Caregiving** - Check in on elderly relatives easily
